# Review Record

使用说明：

- 这是通用 review 记录模板，可用于代码、文档、skill、提示词或配置评审。
- 可与当前 skill pack 中的 `hf-spec-review`、`hf-design-review`、`hf-tasks-review`、`hf-test-review`、`hf-code-review` 或 `hf-traceability-review` 配合使用。
- **默认保存路径：`features/<active>/reviews/<kind>-<scope>-YYYY-MM-DD.md`**（任务级 review 用 `<kind>-review-task-NNN.md`，全 feature 一次性 review 可省略 scope，例如 `traceability-review.md`）。
- 若项目在 `AGENTS.md` 中声明了等价路径，优先遵循项目约定。
